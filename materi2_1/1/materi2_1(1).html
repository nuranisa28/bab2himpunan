F = {x | 7 < x < 9, x ∈ bilangan ganjil}
Apakah himpunan F memiliki anggota?
Yup! Benar sekali!
Ups! Jawabanmu masih keliru.

 Himpunan F tidak memiliki anggota. Mengapa? Coba perhatikan kembali pada himpunan F, apakah ada bilangan ganjil di antara 7 dan 9? Sedangkan yang telah kita pelajari, bilangan ganjil sehabis 7 adalah 9.

Dengan demikian, dapat dikatakan bahwa himpunan F merupakan Himpunan Kosong, karena himpunannya tidak memiliki anggota dan dapat dinotasikan :
F = { } atau F = ∅

Dari kegiatan mengamati tadi, didapatkan pengertian Himpunan Kosong, yaitu :
Himpunan kosong adalah himpunan yang tidak memiliki anggota dan dinotasikan dengan { } atau ∅.
Jika himpunan A merupakan himpunan kosong, maka himpunan A tidak memiliki anggota. Dengan kata lain A = { } atau A = ∅.
