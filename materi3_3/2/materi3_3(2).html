Himpunan kuasa dari A = {2,3} adalah ....

Penyelesaian :
  1. Tentukan terlebih dahulu himpunan bagian yang mungkin dibuat dari A = {2,3}
    Himpunan bagian dengan : Himpunan
    0 anggota { }
    1 anggota {2}, {3}
    2 anggota {2,3}
    Didapatkan himpunan bagian dari A = {2,3} adalah { }, {2}, {3}, dan {2,3}
  2. Sehingga, himpunan kuasa dari himpunan A adalah P(A) = {{ }, {2}, {3}, {2,3}}
Himpunan kuasa dari B = {warna lampu lalu lintas} adalah ....

Penyelesaian :
  1. Anggota dari himpunan B adalah merah, kuning, dan hijau. Sehingga B = {merah, kuning, hijau} dengan n(B) = 3
  2. Tentukan himpunan bagian yang mungkin dibuat dari B = {merah, kuning, hijau}
    Himpunan bagian dengan : Himpunan
    0 anggota { }
    1 anggota {merah}, {kuning}, {hijau}
    2 anggota {merah,kuning}, {merah,hijau}, {kuning,hijau}
    3 anggota {merah,kuning,hijau}
    Didapatkan himpunan bagian dari B = {merah,kuning,hijau} adalah { }, {merah}, {kuning}, {hijau}, {merah,kuning}, {merah,hijau}, {kuning,hijau}, dan {merah,kuning,hijau}
  3. Sehingga, himpunan kuasa dari himpunan B adalah P(B) = {{ }, {merah}, {kuning}, {hijau}, {merah,kuning}, {merah,hijau}, {kuning,hijau}, {merah,kuning,hijau}}
Himpunan kuasa dari C = {x | x < 5, x ∈ bilangan asli} adalah ....

Penyelesaian :
  1. Anggota dari himpunan C adalah 1, 2, 3, dan 4. Sehingga C = {1, 2, 3, 4} dengan n(C) = 4
  2. Tentukan himpunan bagian yang mungkin dibuat dari C = {1, 2, 3, 4}
    Himpunan bagian dengan : Himpunan
    0 anggota { }
    1 anggota {1}, {2}, {3}, {4}
    2 anggota {1,2}, {1,3}, {1,4}, {2,3}, {2,4}, {3,4}
    3 anggota {1,2,3}, {1,2,4}, {1,3,4}, {2,3,4}
    4 anggota {1,2,3,4}
    Didapatkan himpunan bagian dari C = {1,2,3,4} adalah { }, {1}, {2}, {3}, {4}, {1,2}, {1,3}, {1,4}, {2,3}, {2,4}, {3,4}, {1,2,3}, {1,2,4}, {1,3,4}, {2,3,4}, dan {1,2,3,4}
  3. Sehingga, himpunan kuasa dari himpunan C adalah P(C) = {{ }, {1}, {2}, {3}, {4}, {1,2}, {1,3}, {1,4}, {2,3}, {2,4}, {3,4}, {1,2,3}, {1,2,4}, {1,3,4}, {2,3,4}, dan {1,2,3,4}}
